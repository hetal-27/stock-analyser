# Stock Price Analyser

A Python project that fetches real stock data and visualises 
price trends using moving averages.

## What it does
- Downloads historical stock prices using Yahoo Finance
- Plots the closing price over time
- Adds 20-day and 50-day moving averages to spot trends

## Libraries used
- yfinance
- matplotlib

## How to run
1. Install libraries: pip3 install yfinance matplotlib
2. Run the script: python3 test.py
3. Open the chart: open apple_chart.png

## What I learned
- Fetching real financial data with Python
- Understanding moving averages as trend indicators
- Plotting and visualising data with matplotlib
